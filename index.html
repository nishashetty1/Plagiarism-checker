About LCS Algorithm

The Longest Common Subsequence (LCS) algorithm finds the longest subsequence common to two sequences. Unlike substrings, subsequences are not required to occupy consecutive positions within the original sequences.

How It Works

  • The algorithm uses dynamic programming to build a table that stores the lengths of the longest common subsequences for different segments of the input sequences.
  • It backtracks through the table to find the actual LCS.
  • The resulting length and the sequence itself are then displayed to the user.

Tool Functionality

  • You need to upload 2 text files.
  • The program reads the input, processes it to find the LCS, calculates a similarity score, and updates the display with the results.
  • Highlighted segments show which parts of the texts are similar based on the LCS.
